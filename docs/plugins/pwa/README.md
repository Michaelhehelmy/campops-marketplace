# PWA Plugin

> ⚠️ **Plugin-Specific Documentation** — PWA (Progressive Web App) plugin.

## Overview

Progressive Web App support — install banners, push notification subscriptions, and page-view analytics.

## Features

- **Install Banners** — PWAInstallBanner and PWAInstallButton for home screen install prompts
- **Push Subscriptions** — Manage push notification subscriptions via `plugin_pwa_subscriptions` table
- **Settings Management** — PWA configuration stored in `plugin_pwa_settings` table
- **Page-View Analytics** — Tracks page loads via `listing.public_page_loaded` hook
- **Admin Settings Page** — PWASettingsPage for configuring PWA behavior
