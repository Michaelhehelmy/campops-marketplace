# Financial Operations Plugin

> ⚠️ **Plugin-Specific Documentation** — Financial Operations plugin.

## Overview

Manages commissions, billing folios, charges, and payments for property accounting.

## Features

- **Commissions** — Auto-create 10% commission on booking
- **Guest Folios** — Per-booking running account of charges
- **Posting Charges** — Room, F&B, services
- **Payments** — Apply payments to folio
- **Adjustments** — Credits, voids, corrections
