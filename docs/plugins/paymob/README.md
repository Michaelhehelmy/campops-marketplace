# Paymob Plugin

> ⚠️ **Plugin-Specific Documentation** — Paymob payment plugin.

## Overview

Egyptian payment gateway integration for cards, wallets, and installments.

## Features

- **Card Payments** — Visa, Mastercard
- **Mobile Wallets** — Vodafone Cash, Etisalat, Orange
- **Installments** — Bank installment plans
- **Webhook Integration** — HMAC-verified transaction callbacks
- **Transaction History** — Stored in plugin table
