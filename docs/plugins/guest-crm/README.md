# Guest CRM Plugin

> ⚠️ **Plugin-Specific Documentation** — Guest CRM plugin.

## Overview

Guest CRM provides guest profile views, segment analysis, and statistics built on core user and reservation data.

## Features

- **Guest Profiles** — Unified view of guest data from reservations
- **Segments** — Group guests by booking patterns and behavior
- **Statistics** — CRM aggregate metrics and trends
- **Auto-Profiles** — Guest profiles updated automatically via reservation hooks

## Data Source

This plugin uses core `users` and `reservations` tables — no dedicated plugin tables.
