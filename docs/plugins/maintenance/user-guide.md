# Maintenance Plugin User Guide

> ⚠️ **Plugin-Specific Guide** — How to use the Maintenance plugin.

## For Maintenance Staff

### Daily Workflow

#### Viewing Your Tasks

1. **Login** to staff dashboard
2. Navigate to **Maintenance**
3. See assigned work orders
4. Filter by:
   - Priority (Emergency first!)
   - Category
   - Location
   - Due date

#### Emergency Requests

🔴 **Emergency = Immediate action required**

Examples:
- Power outage
- Major water leak
- Security issue
- Safety hazard

**Process:**
1. Receive notification (SMS/app/pager)
2. Acknowledge immediately
3. Respond on-site ASAP
4. Update status with notes
5. Mark complete when resolved

#### Standard Work Orders

**Starting a Task:**
1. Find work order in your list
2. Click **Start Work**
3. Status changes to "In Progress"
4. Add notes as you work

**During Work:**
- Log parts used (scan barcode if available)
- Record time spent
- Add progress photos if required
- Note any complications

**Completing:**
1. Click **Complete**
2. Enter resolution description
3. Log actual costs
4. Upload completion photos
5. Update room status (if guest room)

### Using Mobile App

If available:

1. **Scan room QR** to see maintenance history
2. **Photo documentation** — Before/after required
3. **Parts lookup** — Check inventory on-site
4. **Offline mode** — Works without WiFi, syncs later

## For Housekeeping

### Reporting Issues

When you find problems during cleaning:

1. **Click Report Issue** (on task or room status)
2. **Select Category:**
   - Broken item
   - Not working
   - Needs repair
   - Safety concern
3. **Add Details:**
   - What's wrong
   - When you noticed it
   - Any guest impact
4. **Add Photo** (strongly recommended)
5. **Submit**

**Issue is automatically:**
- Created as maintenance request
- Assigned based on category
- Flagged if guest room affected

### Tracking Your Reports

View status of issues you reported:

1. Go to **My Reports**
2. See all your submitted issues
3. Track:
   - Current status
   - Who is assigned
   - Expected completion
   - Resolution notes

### Room Status

Mark rooms needing maintenance:

1. During task completion
2. If issues found → **Flag for Maintenance**
3. Room status changes to "Maintenance"
4. Front desk sees room unavailable
5. Next guest cannot be assigned

## For Managers

### Managing Work Orders

**Dashboard View:**
- Total open requests
- Emergency/urgent count
- Overdue items
- Cost summary (month to date)

**Assignment:**
1. View unassigned requests
2. Consider:
   - Staff skills (electrical, plumbing, etc.)
   - Current workload
   - Location proximity
3. Click **Assign** and select staff
4. Set priority if needed
5. Add due date

### Preventive Maintenance

**Schedule View:**

1. **Calendar** showing upcoming preventive tasks
2. Filter by:
   - Asset type
   - Frequency
   - Assigned staff
3. Click task to see:
   - Checklist
   - History
   - Parts needed

**Overdue Tasks:**

- Highlighted in red
- May affect warranty
- Can cause guest complaints
- Escalate if repeatedly missed

### Cost Management

**Track Expenses:**

1. **Parts Inventory**
   - Current stock levels
   - Reorder alerts
   - Usage history

2. **Labor Costs**
   - Hours by staff member
   - Cost by category
   - Budget vs actual

3. **Vendor Management**
   - External contractors
   - Invoice tracking
   - Preferred vendors list

### Analytics

**Reports Available:**

1. **Response Time** — How quickly emergencies handled
2. **Completion Rate** — % of tasks completed on time
3. **Recurring Issues** — Same problem, same location
4. **Cost Trends** — Rising costs may indicate aging assets
5. **Guest Impact** — Maintenance affecting guest experience

## For Front Desk

### Guest Complaints

When guest reports issue:

1. **Listen and empathize**
2. **Create maintenance request:**
   - Go to **Maintenance → New Request**
   - Guest reports = High priority
   - Mark "affectsGuests: true"
   - Add guest room number
3. **Set expectations:**
   - "Maintenance will be there within 30 minutes"
   - Offer compensation if severe
4. **Follow up** — Check if resolved satisfactorily

### Room Availability

**Check Before Assigning:**

1. View room status
2. **Green** = Available
3. **Red/Maintenance** = Do not assign
4. **Yellow** = Check expected completion time

### Communication

**Keep Guests Informed:**

- "Maintenance is on their way"
- "Should be resolved by 3pm"
- "We've moved you to Room 8"
- "Please accept our apologies and this credit"

## Tips & Best Practices

### For Maintenance Staff

- **Prioritize emergencies** — Guest safety first
- **Document everything** — Photos, time, parts used
- **Communicate delays** — If taking longer than expected
- **Keep workspace clean** — Especially in guest areas
- **Test after repair** — Ensure problem actually fixed

### For Housekeeping

- **Report immediately** — Don't wait until end of shift
- **Be specific** — "AC not cooling" not "AC broken"
- **Take photos** — Helps maintenance diagnose
- **Check room after** — Verify maintenance completed

### For Managers

- **Monitor response times** — Especially emergencies
- **Track recurring issues** — May need replacement not repair
- **Budget for preventive** — Cheaper than emergency fixes
- **Cross-train staff** — Reduces dependency on single person
- **Review costs monthly** — Catch budget overruns early

### For Front Desk

- **Never promise room** until status shows "Clean"
- **Log all complaints** — Even if minor
- **Follow up every time** — Close the loop with guest
- **Empower compensation** — Within policy limits

## Troubleshooting

### Emergency Not Responded To

1. Check if staff notified (app/SMS/pager)
2. Verify staff on duty
3. Escalate to manager immediately
4. Consider external contractor

### Guest Unhappy With Repair

1. Apologize sincerely
2. Escalate to supervisor
3. Offer appropriate compensation
4. Reopen work order with notes
5. Consider room change

### Part Not in Stock

1. Check if other locations have part
2. Expedite order if guest-affected
3. Update guest on timeline
4. Consider temporary workaround
5. Source locally if critical

### Work Order Stuck "In Progress"

1. Contact assigned staff
2. Check for complications
3. Reassign if staff unavailable
4. Update guest on status
5. Escalate if overdue
