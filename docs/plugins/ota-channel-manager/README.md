# OTA Channel Manager Plugin

> ⚠️ **Plugin-Specific Documentation** — OTA Channel Manager plugin.

## Overview

Syncs availability and bookings with Online Travel Agencies (Booking.com, Expedia, Airbnb).

## Features

- **External Calendar Management** — Create and list ICS calendar URLs per room
- **Booking Conflict Detection** — View reservations with `status = 'conflict_pending'`
- **Auto Sync** — Registers a `cron.hourly` stub handler for periodic sync
