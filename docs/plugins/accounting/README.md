# Accounting Plugin

> ⚠️ **Plugin-Specific Documentation** — Accounting plugin.

## Overview

Ledger entries, invoices, expenses, and revenue summary.

## Features

- **Ledger Entries** — Record financial transactions
- **Invoices** — Manage invoice records
- **Expenses** — Track expense items
- **Revenue Summary** — Aggregated revenue view
