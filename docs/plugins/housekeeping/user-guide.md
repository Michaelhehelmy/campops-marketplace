# Housekeeping Plugin User Guide

> ⚠️ **Plugin-Specific Guide** — How to use the Housekeeping plugin.

## For Housekeeping Staff

### Daily Workflow

#### Starting Your Shift

1. **Login** to staff dashboard
2. View **My Tasks** for today
3. Tasks show:
   - Room number
   - Task type (checkout clean, stayover, etc.)
   - Priority level
   - Estimated time
   - Special notes

#### Working Tasks

**To start a task:**
1. Find task in your list
2. Click **Start Task**
3. Room status changes to "In Progress"

**While cleaning:**
- Follow standard checklist:
  - Change linens
  - Clean bathroom
  - Restock amenities
  - Vacuum/sweep
  - Check for damage

**To complete a task:**
1. Click **Complete**
2. Mark any issues found
3. Room status changes to "Clean" or "Inspected"

#### Task Priority Levels

- 🔴 **Urgent** — VIP checkout, immediate turnaround
- 🟠 **High** — Standard checkout, next guest arriving soon
- 🟡 **Normal** — Stayover service, non-urgent cleaning
- 🟢 **Low** — Deep cleaning, maintenance when available

### Using Mobile App

If available on your property:

1. **Download** staff app
2. **Login** with staff credentials
3. **Scan QR code** on room door to auto-load task
4. **Update status** directly from room
5. **Add photos** of completed work (if required)

## For Managers

### Managing Staff Workload

**View Staff Dashboard:**
- **Staff → Housekeeping**
- See tasks assigned to each staff member
- View completion rates
- Check current workload

**Redistribute Tasks:**
1. Select staff member with light load
2. Click **Assign Tasks**
3. Select unassigned or overloaded staff tasks
4. Confirm reassignment

### Quality Control

#### Inspection Process

If inspection is enabled:

1. **Staff completes** cleaning task
2. Room marked **"Clean - Pending Inspection"**
3. Inspector receives notification
4. Inspector reviews room:
   - Pass → Room marked **"Inspected - Ready"**
   - Fail → Returned to staff with notes

#### Spot Checks

Even without full inspection:

1. Randomly select cleaned rooms
2. Use **Quick Inspect** feature
3. Rate cleanliness 1-5
4. Provide feedback to staff

### Reporting Issues

**During Cleaning:**

If staff finds maintenance issues:

1. Click **Report Issue** on task
2. Select issue type:
   - Broken fixture
   - Plumbing problem
   - Electrical issue
   - Damage
3. Add photo if possible
4. Mark priority

**Issue automatically:**
- Creates maintenance task
- Notifies maintenance staff
- Flags room as "Maintenance" status

### Analytics

**View Reports:**

1. **Reports → Housekeeping**
2. Available metrics:
   - Average cleaning time by room type
   - Staff productivity
   - Tasks completed on time
   - Quality scores
   - Issue frequency by room

## For Front Desk

### Checking Room Status

**View Real-time Status:**

1. **Dashboard → Room Status**
2. See all rooms:
   - 🟢 **Clean/Ready** — Available for check-in
   - 🟡 **In Progress** — Being cleaned
   - 🔴 **Dirty** — Needs cleaning
   - ⚪ **Inspected** — Ready (with inspection)
   - 🔧 **Maintenance** — Not available

**Filter by:**
- Floor/area
- Room type
- Status
- Expected checkout

### Handling Early Check-ins

1. Guest arrives early
2. Check room status
3. If room is **In Progress**:
   - Contact housekeeping
   - Request rush cleaning
   - Offer waiting area
4. If room is **Dirty**:
   - Check if any clean rooms available
   - Offer upgrade if applicable

### Late Checkouts

When guest requests late checkout:

1. **Update booking** with new checkout time
2. **Housekeeping notified** automatically
3. Task creation delayed accordingly
4. Next guest informed of potential delay

## Tips & Best Practices

### For Staff

- **Start with urgent tasks** — VIP rooms and early arrivals
- **Report issues immediately** — Don't wait until task complete
- **Follow the checklist** — Ensures consistency
- **Restock amenities** — Check shampoo, soap, towels
- **Check for guest items** — Lost & found procedure

### For Managers

- **Balance workloads** — Don't overload single staff member
- **Recognize good work** — High completion rates, quality scores
- **Address issues quickly** — Maintenance problems, supply shortages
- **Schedule deep cleans** — During low occupancy periods
- **Cross-train staff** — Backup coverage for absences

### For Front Desk

- **Check room status** before promising early check-in
- **Communicate delays** to waiting guests proactively
- **Update booking system** for late checkouts
- **Coordinate with housekeeping** for VIP arrivals

## Troubleshooting

### Room Shows Clean But Isn't

1. Contact housekeeping supervisor
2. Reassign cleaning task
3. Update room status to "Dirty"
4. Document for quality tracking

### Staff Can't Access Tasks

1. Verify staff login credentials
2. Check shift assignment
3. Confirm housekeeping role is assigned
4. Try refreshing dashboard

### Tasks Not Auto-Creating

1. Verify checkout was processed in system
2. Check auto-create settings in plugin config
3. Manually create task if needed
4. Report to manager for investigation

### Running Behind Schedule

1. Identify critical rooms (next arrivals)
2. Request additional staff if available
3. Contact guests with early check-ins
4. Consider offering compensation for delays
