# Booking Plugin User Guide

> ⚠️ **Plugin-Specific Guide** — This guide covers using the Booking plugin features.

## For Property Managers

### Managing Bookings

#### Viewing All Bookings

1. Go to **Dashboard → Bookings**
2. See a list of all bookings with status indicators:
   - 🟡 **Pending** — Awaiting payment or confirmation
   - 🟢 **Confirmed** — Booking is active
   - 🔵 **Checked-in** — Guest has arrived
   - ⚪ **Checked-out** — Guest has departed
   - 🔴 **Cancelled** — Booking cancelled

#### Creating a Manual Booking

For phone/walk-in bookings:

1. Click **New Booking** button
2. Select dates and room type
3. Enter guest information
4. Choose payment method:
   - **Pay Now** — Guest pays immediately
   - **Pay at Property** — Collect on arrival
   - **Invoice** — Send payment link via email
5. Click **Create Booking**

#### Modifying Bookings

1. Click on a booking to open details
2. Use the **Actions** menu:
   - **Edit Dates** — Change check-in/out dates
   - **Change Room** — Move to different room
   - **Add Guests** — Update guest count
   - **Update Status** — Check-in, check-out, cancel
   - **Process Refund** — Issue partial or full refund

#### Check-in Process

1. Find booking in list (filter by "Confirmed")
2. Click **Check In** button
3. Verify guest ID
4. Collect any remaining payment
5. Provide room key/access info
6. Mark as **Checked-in**

#### Daily Operations

**Arrivals Today:**
- Dashboard shows guests checking in today
- Prepare rooms in advance
- Send welcome messages automatically

**Departures Today:**
- Guests checking out today
- Schedule housekeeping after checkout
- Process any late checkout requests

**Overstays:**
- System alerts if guest hasn't checked out by deadline
- Contact guest or extend booking

### Managing Room Inventory

#### Setting Availability

1. Go to **Settings → Rooms**
2. Click on a room type
3. Use calendar view to block/unblock dates
4. Or bulk update: **Block Dates** button

#### Seasonal Pricing

1. Navigate to **Pricing → Seasonal Rates**
2. Create new rate period
3. Set:
   - Date range
   - Price multiplier (e.g., 1.5 for 50% increase)
   - Days of week (optional)
4. Save and apply

### Reports

#### Occupancy Report

**Reports → Occupancy**

Shows:
- Occupancy rate by date
- Revenue per available room
- Average daily rate
- Compare to previous periods

#### Revenue Report

**Reports → Revenue**

Breaks down:
- Total bookings
- Revenue by room type
- Cancellation revenue lost
- Average booking value

## For Staff

### Front Desk Operations

#### Quick Booking Lookup

Use the search bar in the bookings list:
- Search by guest name
- Search by booking ID
- Search by phone number
- Search by email

#### Today's Dashboard

The main dashboard shows:
- **Arrivals** — Guests checking in today
- **Departures** — Guests leaving today
- **Stayovers** — Guests staying another night
- **Pending Check-ins** — Confirmed but not yet arrived

#### Handling Walk-ins

1. Check availability for today
2. Create new booking (see manager section)
3. Process immediate payment if needed
4. Assign room and provide key

### Guest Communication

#### Automated Messages

The system automatically sends:
- **Booking Confirmation** — Immediately after booking
- **Payment Reminder** — 24h before deadline
- **Check-in Reminder** — 24h before arrival
- **Check-out Reminder** — Morning of departure

#### Manual Messages

1. Open booking details
2. Click **Message Guest** button
3. Choose template or write custom message
4. Send via email or SMS (if guest provided phone)

## For Guests

### Making a Booking

#### Online Booking Process

1. Visit property website
2. Enter dates and guest count
3. Browse available rooms
4. Select room and click **Book Now**
5. Enter guest information:
   - Name
   - Email (required)
   - Phone
   - Special requests
6. Review booking details
7. Complete payment (if required)
8. Receive confirmation email with booking reference

#### Booking Confirmation

Your confirmation email includes:
- Booking reference number
- Property address and contact
- Check-in/out times
- Room details
- Total amount paid
- Cancellation policy
- Directions/map link

#### Modifying Your Booking

Guests can modify via:

1. **Confirmation Email** — Click "Manage Booking" link
2. **Property Website** — Find "My Booking" section
3. **Contact Property** — Call or email with booking reference

Modifications allowed:
- Change dates (subject to availability)
- Change room type
- Update guest count
- Add special requests

#### Cancellation

To cancel:

1. Use "Manage Booking" link in confirmation email
2. Or contact property directly
3. Cancellation policy applies:
   - 48+ hours before: Full refund
   - 24-48 hours: 50% refund
   - Less than 24 hours: No refund

### During Your Stay

#### Check-in

- Arrive during check-in hours (typically 3pm)
- Present ID and booking confirmation
- Receive room key/access information
- Get property amenities info

#### During Stay

- Access guest portal (link in confirmation email)
- View property map
- See amenity hours
- Request housekeeping
- Extend stay (if available)

#### Check-out

- Depart by check-out time (typically 11am)
- Return keys
- Request late checkout in advance if needed
- Receive receipt via email

## Tips & Best Practices

### For Managers

- **Enable auto-confirm** for paid bookings to reduce manual work
- **Set realistic hold times** (15 min is standard)
- **Use seasonal pricing** to maximize revenue
- **Monitor cancellation patterns** and adjust policy if needed
- **Integrate with housekeeping** for automatic task creation

### For Staff

- Always verify guest ID at check-in
- Check for special requests before arrival
- Keep front desk stocked with maps and property info
- Have backup rooms available for emergencies

### For Guests

- Book early for peak season
- Read cancellation policy before booking
- Save booking reference number
- Arrive on time or notify property of delays
- Check out on time or request extension in advance

## Troubleshooting

### Can't Find My Booking

- Check spam folder for confirmation email
- Contact property with:
  - Name used for booking
  - Email address
  - Approximate booking date
  - Phone number

### Payment Failed

- Try different payment method
- Ensure card has sufficient funds
- Contact bank if card is declined
- Property can hold booking for 24h while resolving

### Need to Cancel

- Check cancellation deadline in confirmation email
- Cancel ASAP for best refund
- Contact property directly for urgent cancellations

### Room Not Ready

- Standard check-in time applies
- Early check-in may be available (contact property)
- Property will store luggage if room isn't ready
