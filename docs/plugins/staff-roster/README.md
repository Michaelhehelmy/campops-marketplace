# Staff Roster Plugin

> ⚠️ **Plugin-Specific Documentation** — This documentation is for the Staff Roster plugin.

## Overview

Manages staff schedules, shift assignments, time tracking, and labor cost reporting.

## Features

- **Staff Listing** — List staff for a property by listingId
- **Shift Scheduling** — Create and list staff shifts by date range
- **Roster Views** — Filter shifts by start/end date range
