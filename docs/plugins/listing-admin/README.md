# Listing Admin Plugin

> ⚠️ **Plugin-Specific Documentation** — Listing Admin plugin.

## Overview

Provides property performance analytics and dashboard widgets built on core reservation and listing data.

## Features

- **Property Performance Widget** — Dashboard widget showing revenue, fees, bookings, and net payout
- **Dashboard Stats Hook** — Listens to `dashboard.get_stats` to compute listing analytics
- **No Database Tables** — Reads from core tables, no dedicated plugin storage
