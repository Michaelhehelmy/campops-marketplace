# Loyalty Plugin User Guide

> ⚠️ **Plugin-Specific Guide** — How to use the Loyalty rewards program.

## For Property Managers

### Managing the Program

#### Program Overview

**Dashboard shows:**
- Total enrolled members
- Points issued this month
- Points redeemed this month
- Liability (outstanding points value)
- Top members by tier

#### Enrolling Guests

**Automatic Enrollment:**
- Guests auto-enroll on first booking
- Welcome bonus applied immediately
- Confirmation email sent

**Manual Enrollment:**
1. **Loyalty → Enroll Guest**
2. Search for guest profile
3. Click **Enroll**
4. Optional: Add welcome bonus points
5. Send welcome email

**Bulk Enrollment:**
1. Export guest list
2. Use bulk import tool
3. Set initial tiers based on history

#### Managing Tiers

**View Members by Tier:**

1. **Loyalty → Members**
2. Filter by tier:
   - Bronze (new members)
   - Silver (active members)
   - Gold (frequent guests)
   - Platinum (VIP)

**Manual Tier Override:**

1. Find member
2. Click **Edit**
3. Change tier manually
4. Add reason note
5. Save

*Note: Manual tiers don't affect point multipliers unless configured*

#### Points Adjustments

**Add/Remove Points:**

1. Find member
2. Click **Adjust Points**
3. Enter amount (positive or negative)
4. Add reason:
   - "Service recovery"
   - "Promotional bonus"
   - "Correction"
5. Save

**Bulk Point Grant:**

For promotions (e.g., "Everyone gets 500 bonus points"):

1. **Loyalty → Promotions**
2. Select "Bulk Point Grant"
3. Set filters (all members, specific tier, etc.)
4. Enter point amount
5. Add reason
6. Confirm and send

### Promotions

#### Creating a Double Points Event

1. **Loyalty → Promotions → New**
2. Select type: "Double Points"
3. Set dates
4. Choose applies to:
   - All bookings
   - Specific room types
   - Specific dates
5. Add promo code (optional)
6. Save and activate

#### Birthday Rewards

System automatically:
- Sends birthday email
- Awards bonus points
- Offer special rate

Configure in **Loyalty → Settings → Birthday**

### Reporting

**Available Reports:**

1. **Program Performance**
   - Enrollment growth
   - Points issued/redeemed
   - Breakage (expired points)

2. **Member Activity**
   - Average points balance
   - Time to redeem
   - Tier movement

3. **ROI Analysis**
   - Revenue from loyalty members
   - Incremental bookings
   - Program cost vs benefit

## For Front Desk Staff

### Checking Guest Loyalty Status

**During Check-in:**

1. Booking screen shows loyalty badge
2. Click to see:
   - Current tier
   - Points balance
   - Available benefits
   - Redemption history

**Loyalty Benefits to Mention:**

- "Welcome back Mr. Smith! As a Gold member, you're eligible for a room upgrade."
- "I see you have 2,500 points. Would you like to redeem $25 toward this stay?"
- "As a Silver member, you get late checkout until 1pm."

### Processing Redemptions

**Guest Requests to Redeem:**

1. **Verify balance** — Ensure sufficient points
2. **Apply redemption:**
   - Discount: Apply code to booking
   - Upgrade: Modify room assignment
   - Perk: Note in booking (late checkout, etc.)
3. **Confirm with guest:**
   - "I've applied your $50 discount using 500 points."
   - "You've been upgraded to a suite using 1,000 points."
4. **Points deducted automatically**

**Redemption at POS:**

1. In POS screen, click **Loyalty**
2. Scan guest QR or enter phone/email
3. Show available redemptions
4. Guest selects reward
5. Apply to current bill

### Enrolling Walk-ins

**Guest Wants to Join:**

1. **Loyalty → Quick Enroll**
2. Enter:
   - Name
   - Email (required)
   - Phone
3. **Send welcome kit:**
   - Email with member number
   - Program details
   - Current promotions
4. **Award welcome bonus** immediately

## For Guests

### Joining the Program

**How to Enroll:**

1. **Book a stay** — Auto-enrolled on first booking
2. **Online** — Website loyalty page
3. **At property** — Ask front desk
4. **Via email** — Click link in marketing email

**Welcome Benefits:**
- Instant Bronze tier
- 100-500 welcome points
- Member-only rates
- Exclusive promotions

### Earning Points

**How Points Work:**

- **Bookings:** 1-2 points per $1 spent (tier multiplier applies)
- **On-site purchases:** Restaurant, spa, activities earn points
- **Bonus events:**
  - First booking: 500 bonus
  - Write review: 100 bonus
  - Refer friend: 1,000 bonus
  - Birthday month: 200 bonus

**Viewing Your Balance:**

1. **Login** to member portal
2. See:
   - Current points
   - Lifetime points
   - Tier status
   - Recent activity
3. **Mobile app** — Check anytime

### Redeeming Points

**What You Can Get:**

| Reward | Points | Value |
|--------|--------|-------|
| $25 Discount | 250 | $25 |
| $50 Discount | 500 | $50 |
| Room Upgrade | 1,000 | Variable |
| Late Checkout | 300 | Convenience |
| Free Night | 5,000 | Room rate |

**How to Redeem:**

**Online Booking:**
1. Search and select room
2. At checkout, click **Use Points**
3. Select reward
4. Points deducted, discount applied

**At Check-in:**
1. Tell front desk: "I'd like to redeem points"
2. Staff checks your balance
3. Choose reward
4. Applied immediately

**At Restaurant/Spa:**
1. Ask staff: "Can I use loyalty points?"
2. Show member number or QR code
3. Select redemption amount
4. Deducted from bill

### Tier Benefits

**Bronze** (0+ points):
- Member-only rates
- Earn points on stays

**Silver** (1,000+ points):
- All Bronze benefits
- Late checkout (1pm)
- 25% bonus points
- Free WiFi upgrade

**Gold** (5,000+ points):
- All Silver benefits
- Room upgrades (subject to availability)
- Late checkout (2pm)
- 50% bonus points
- 10% booking discount
- Priority support

**Platinum** (10,000+ points):
- All Gold benefits
- Suite upgrades
- Late checkout (4pm)
- 100% bonus points (double points)
- 20% booking discount
- Free breakfast
- Dedicated concierge

### Referring Friends

**How Referrals Work:**

1. **Get your referral link:**
   - Member portal → Refer Friends
   - Or: "Share and Earn" in mobile app

2. **Share with friends:**
   - Email, text, social media
   - Each friend gets unique code

3. **Friend books:**
   - Uses your referral code
   - Completes first stay

4. **You both win:**
   - You get 1,000 points
   - Friend gets 500 welcome points

**Track Referrals:**

- See pending referrals
- Track who booked
- View points earned from referrals

## Tips & Best Practices

### For Managers

- **Promote actively** — Mention program in all guest communications
- **Train staff** — Ensure everyone knows how to check/enroll
- **Monitor liability** — Points are a financial liability
- **Set expiration** — Encourage redemption, manage liability
- **Target Gold/Platinum** — VIP treatment drives loyalty

### For Staff

- **Always ask** — "Are you a loyalty member?"
- **Mention benefits** — Upsell tier benefits during interactions
- **Help redeem** — Make it easy, don't make guests figure it out
- **Suggest enrollment** — "Would you like to join? It's free and you get 500 points today."

### For Guests

- **Book direct** — Often better rates and full points
- **Use your points** — They can expire, don't hoard indefinitely
- **Refer friends** — Easy way to earn big point bonuses
- **Check promotions** — Double points events multiply quickly
- **Redeem strategically** — Save points for high-value redemptions

## Troubleshooting

### Didn't Get Points for Stay

1. Stay must be completed (not just booked)
2. Direct bookings earn more than OTAs
3. Some rates excluded (group bookings, corporate)
4. Contact property with booking number

### Can't Redeem Points

- Need sufficient points for chosen reward
- Some rewards have blackout dates
- Verify account is in good standing
- Points may be pending (recently earned)

### Tier Benefits Not Applied

- Benefits apply at check-in
- Subject to availability (upgrades)
- May need to remind staff of tier status
- Some benefits require advance notice

### Missing Referral Bonus

- Friend must use your unique code
- Friend must complete first stay
- Bonus applies after checkout
- Max 5 referrals per month

### Account Issues

**Merge Accounts:**
- Created duplicate account?
- Contact support to merge
- Points and history combined

**Missing History:**
- Older stays may not have imported
- Contact property with old booking numbers
- Can manually add with proof of stay
